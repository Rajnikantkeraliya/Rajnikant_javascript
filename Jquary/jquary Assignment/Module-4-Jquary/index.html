1 .What is jQuery?

⤖ jQuery is a fast, small, and feature-rich JavaScript library.

⤖It simplifies various tasks such as HTML document traversal and manipulation, event handling, animation, and AJAX (Asynchronous JavaScript and XML) interactions.

2 .How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery , JQuery Animation?

⤖ To apply CSS using jQuery

// Applying CSS using jQuery

$(document).ready(function(){

// Selecting elements by class and applying CSS

$(".myElement").css("color", "red");

// Selecting elements by ID and applying CSS

$("#myElement").css({

"font-size": "20px",

"background-color": "blue"});

});

⤖Adding and removing classes using jQuery

$(document).ready(function(){

// Adding a class

$(".myElement").addClass("highlight");

// Removing a class

$("#myElement").removeClass("inactive");

});

⤖jQuery animation

$(document).ready(function(){

// Animating an element's width

$("#myElement").animate({ width: "500px"}, 1000); // 1000 milliseconds (1 second) duration // Sliding an element down

$(".slideDownButton").click(function(){ $(".slideDownElement").slideDown(); });

// Fading an element out

$(".fadeOutButton").click(function(){ $(".fadeOutElement").fadeOut(); });

});
