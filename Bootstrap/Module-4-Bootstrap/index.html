1 What are the advantages of Bootstrap?

Rapid Development:

⤖ Bootstrap provides pre-designed components and styles, enabling quick and easy website development.

Responsive Design:

⤖ Built-in responsiveness ensures websites look and function well across various devices and screen sizes.

Consistency:

⤖ Bootstrap's standardized components and grid system maintain visual consistency throughout the website.

Customization:

⤖ Easy to customize with pre-defined classes or by overriding default styles to suit specific design needs.

Cross-Browser Compatibility:

⤖ Ensures consistent appearance and functionality across different web browsers.

2 .What is a Bootstrap Container, and how does it work?

⤖ A Bootstrap container is a fundamental layout element used to wrap and contain the content within a webpage. It helps to establish the responsive grid system and maintain consistent spacing and alignment.

Container :

⤖ The standard container provides a responsive fixed-width layout. It adapts to different screen sizes but maintains a fixed maximum width.

Container-fluid:

⤖ The container-fluid class creates a full-width container that spans the entire width of the viewport.

3 What are the default Bootstrap text settings?

⤖ Bootstrap, a popular front-end framework, provides default styles for typography to help improve the appearance and readability of text on a website.

Font Family:

⤖ The default font-family is set to 'Helvetica Neue', Helvetica, Arial, sans-serif;.

Font Size:

⤖ Bootstrap sets a base font size of 16px for the body element.

Line Height:

⤖ The default line height is 1.5 to improve readability.

Font Weight:

⤖ The default font weight for most elements is normal (400), but it can vary for different components.

Text Color:

⤖ The default text color is usually set to a dark shade.

Headings:

⤖ Headings (h1 to h6) have different default sizes, ranging from larger (h1) to smaller (h6).

Text Alignment:

⤖ Text is left-aligned by default.

Links:

⤖Links have a default blue color and are underlined. Visited links are styled differently to indicate their visited state.

Text Transformation:

⤖ Uppercase is often applied to certain headings, and text-transform properties might be used in other components.

Spacing:

⤖ Bootstrap sets default margin and padding values to provide consistent spacing between elements.

4.What do you know about the Bootstrap Grid System?

    

⤖ Bootstrap's grid system is built with flexbox and allows up to 12 columns across the page.

⤖ If you do not want to use all 12 columns individually, you can group the columns together to create wider columns:

⤖ The grid system is responsive, and the columns will re-arrange automatically depending on the screen size.

⤖ Make sure that the sum adds up to 12 or fewer (it is not required that you use all 12 available columns).

Grid Classes:-

⤖.col- (extra small devices - screen width less than 576px)

⤖.col-sm- (small devices - screen width equal to or greater than 576px)

⤖.col-md- (medium devices - screen width equal to or greater than 768px)

⤖.col-lg- (large devices - screen width equal to or greater than 992px)

⤖.col-xl- (xlarge devices - screen width equal to or greater than 1200px)

⤖.col-xxl- (xxlarge devices - screen width equal to or greater than 1400px)
